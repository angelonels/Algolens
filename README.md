# AlgoLens

A visual playground for understanding algorithms. Pick an algorithm, watch it run step by step, and finally get why it works.

## What's inside

- Binary Search
- Bubble Sort
- Insertion Sort
- Merge Sort
- Euclidean GCD
- Matrix Traversal

Each one comes with animations, a Python snippet you can copy, and a short explanation.

## Tech

- React 19
- Vite
- React Router
- Framer Motion

## Project structure

## Running locally

```bash
cd AlgoLens
npm install
npm run dev
```

Opens at `http://localhost:5173`
